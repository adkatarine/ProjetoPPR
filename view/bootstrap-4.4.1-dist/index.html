Sobre

Este site é um projeto da disciplina EXA844 - Programação para Redes 2019.2.

Seu objetivo é mostrar ao usuário quão satisfatório tem sido os filmes que estão/estarão em cartaz no cinema através de um ranking. Os dados são extraídos do twitter e assim feita uma análise de sentimentos dos tweets.
